# LIDAR Test Reference

Initial Build 22 guide: Kustom Signals ProLaser 4 - RCMP Configuration.

Official source: https://kustomsignals.com/portal_resources/006-0966-12_REV_3_RCMP.pdf

Manual library: https://kustomsignals.com/service-and-support/operators-manuals

## Scope

The Build 22 local guide covers:

- Training and authorization confirmation.
- Device inspection.
- Self-test.
- HUD alignment for the ProLaser 4 RCMP guide only.
- Differential-distance test for the ProLaser 4 RCMP guide only.

No other LIDAR model receives ProLaser 4 instructions.
