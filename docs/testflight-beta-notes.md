# TestFlight Beta Notes

## Copy for App Store Connect

OPAi Police is currently in internal testing. This build is a local/offline prototype designed to evaluate navigation, workflow structure, UI/UX, disclaimers, local persistence, local notifications, incident drafts, translation mockups, AI Assistant mockups, notes/files metadata, court/training reminders, and settings/legal flows.

This build does not connect to a production backend, OpenAI, police systems, cloud storage, payment services, or real authentication.

Please do not enter real police records, confidential information, sensitive personal information, real evidence, real statements, or official documents.
