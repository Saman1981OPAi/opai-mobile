# Internal Beta Launch Announcement

## Message

OPAi Police internal iOS beta is now ready for testing through TestFlight.

This build is a local/offline prototype designed to test workflow, navigation, UI/UX, reminders,
local persistence, local notifications, mock AI Assistant, mock translation, incident drafts,
notes/files metadata, legal/privacy screens, and App Store readiness.

Please use demo data only. Do not enter real police records, confidential information, sensitive
personal information, real evidence, real statements, or official documents.

## Please Test

- Mock authentication and consent
- Home, Start My Shift, and local incident drafts
- Mock Assistant and translation workflows
- Calendar, Court, Training, Requalification, and Follow-Ups
- Notes & Files, Settings, legal screens, reset/clear, and local notifications

## Do Not Test as Production Functionality

There is no live backend, OpenAI, translation/OCR service, cloud sync, real account, file upload,
evidence storage, police-system integration, payment, or official records-management capability.
Do not rely on the app for official duties, legal/medical advice, emergencies, or professional
decisions.

Submit TestFlight feedback or email support@opaiapp.com with device, iOS version, build, module,
steps, expected result, actual result, and sanitized screenshots when possible.
