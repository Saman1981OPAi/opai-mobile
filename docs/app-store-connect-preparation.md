## App Review Notes Draft

OPAi Police is currently a local/offline testing prototype. The app uses mock authentication, local persistence, local notifications, mock AI Assistant responses, mock translation workflows, local incident drafts, and local notes/file metadata placeholders. The app does not connect to a production backend, OpenAI, police systems, cloud storage, payment services, or real authentication. Please use demo data only.
