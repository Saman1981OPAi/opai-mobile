# Court Reminder Workflow

## Purpose

The Court workflow gives users a local place to organize court reminders during prototype testing.

## Current Sprint 009 Behavior

Court reminder fields:

- Matter name
- Court name
- File reference
- Date
- Time
- Location
- Courtroom
- Notes
- Reminder enabled
- Reminder lead time
- Status: upcoming, completed, adjourned, cancelled

Supported local actions:

- Add
- Edit
- Delete
- Complete/reopen
- Adjourn/cancel through status cycling
- Schedule or cancel local reminders

## Safety Notice

Court reminders are not official court records. Users must verify court dates, file details, courtroom assignments, adjournments, and obligations through authorized systems.
