# Screenshot Correction Procedure

Verify Apple's current App Store Connect specifications before upload. For the current package, use
one of these accepted pixel dimensions.

## iPhone 6.9-inch

- 1260 x 2736 portrait or 2736 x 1260 landscape
- 1290 x 2796 portrait or 2796 x 1290 landscape
- 1320 x 2868 portrait or 2868 x 1320 landscape

## iPad

- 2064 x 2752 portrait or 2752 x 2064 landscape
- 2048 x 2732 portrait or 2732 x 2048 landscape

## Correction Steps

1. Inspect the actual source image dimensions; do not rely on the filename.
2. Re-capture or export at an accepted canvas size without stretching the UI.
3. Save as PNG, JPG, or JPEG using App Store Connect's current format rules.
4. Check safe-area framing, status/profile overlays, crop, text wrapping, and clarity.
5. Confirm the image shows actual current functionality and demo information only.
6. Confirm it contains no protected police/government mark or unlicensed imagery.
7. Re-upload only the affected device set and review the App Store preview.
8. Record the corrected dimensions and source build.

Do not solve screenshot rejection by adding functionality that is absent from the app.
