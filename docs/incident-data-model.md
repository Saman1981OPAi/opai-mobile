# Incident Data Model

## Local Draft Fields

Sprint 010 stores the following local prototype fields:

- Incident type
- Occurrence category
- Date
- Time
- Location
- Priority
- Status
- Follow-up required flag
- Persons involved
- Witnesses
- Officer notes
- Narrative draft
- Observations
- Follow-up notes
- Disclosure notes placeholder
- Attachment metadata
- Created date
- Updated date

## Status Values

- Draft
- In Review
- Follow-up Required
- Reviewed
- Archived

## Priority Values

- Low
- Medium
- High
- Urgent

Priority labels are local productivity labels only and do not replace official dispatch, RMS, supervision, or service policy.

## AI-Ready Structure

The structured draft can later support:

- AI report drafting
- AI report review
- AI follow-up suggestions
- AI summary generation

Sprint 010 does not call OpenAI or generate real AI output. The “Prepare for AI Review” button displays a local placeholder warning only.
