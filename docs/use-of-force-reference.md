# Use of Force Reference

The Use of Force Reference is not a device test. It appears under Operational Reference.

Sources:

- https://www.publicsafety.gc.ca/lbrr/archives/cnmcs-plcng/cn31151-eng.pdf
- https://www.ontario.ca/laws/regulation/r23391

Build 22 uses an original non-linear OPAi visualization:

ASSESS -> PLAN -> ACT -> REASSESS

Surrounding considerations:

- Situation
- Subject behaviour
- Officer perception
- Tactical considerations
- Continuous reassessment

The app must never create a force-selection calculator, threat score, automatic force recommendation, legal-justification generator, weapon recommendation, pass/fail assessment, or linear force ladder.
