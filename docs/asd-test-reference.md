# ASD Test Reference

Full title: Approved Screening Device Check.

Federal source: https://laws-lois.justice.gc.ca/eng/regulations/SOR-2024-66/page-1.html

Initial representative models:

- Alco-Sensor FST
- Drager Alcotest 7000 Canadian Police
- Other device appearing in the current Approved Screening Devices Order receives an unsupported-guide notice unless a verified local guide exists.

The Build 22 app requires manufacturer and exact model selection before showing guide content.
