# File Metadata Limitations

Sprint 013 introduces file metadata placeholders only. The OPAi Police prototype does not upload, store, process, read, scan, classify, parse, sync, or transmit real files.

## Not Included

- Camera access
- Microphone access
- Photo library access
- Document picker access
- File upload
- File download
- Cloud file storage
- Evidence storage
- OCR processing
- Audio transcription
- Video processing
- Police evidence-system integration
- Court/disclosure-system integration

## Intended Use

Metadata placeholders let reviewers see how future file-related workflows may be organized without exposing real documents, evidence, personal information, or police records.

Users must not enter real evidence details, real statements, official documents, confidential information, sensitive personal information, or operational police records in this prototype.
