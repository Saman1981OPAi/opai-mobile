# iOS Screenshot Checklist

## iPhone 6.9-Inch Accepted Sizes

- 1260 x 2736 portrait
- 2736 x 1260 landscape
- 1290 x 2796 portrait
- 2796 x 1290 landscape
- 1320 x 2868 portrait
- 2868 x 1320 landscape

## iPad Accepted Sizes

- 2064 x 2752 portrait
- 2752 x 2064 landscape
- 2048 x 2732 portrait
- 2732 x 2048 landscape

## Checklist

- [ ] Screenshots match actual app functionality.
- [ ] No official police or government insignia.
- [ ] No real personal information.
- [ ] No real police records.
- [ ] No copyrighted third-party content.
- [ ] Testing/pre-launch status is not misleading.
- [ ] PTSD awareness accent uses `#7FFFD4`.
- [ ] Dark blue/black OPAi brand is consistent.
- [ ] Screenshots do not overpromise AI, legal, medical, translation, or operational capabilities.

## Recommended First Screens

- Home Dashboard.
- Start My Shift.
- New Incident or OPAi Assistant.
- Translation.
- Calendar/Court/Training.
- Notes & Files or Settings.
