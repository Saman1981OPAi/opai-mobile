# Post-Upload Monitoring Checklist

## App Store Connect

- [ ] Build processing starts and completes.
- [ ] No binary, icon, permission, signing, or version warning appears.
- [ ] Export compliance, content rights, age rating, and App Privacy are complete.
- [ ] Beta Review status is monitored if applicable.
- [ ] Screenshot, metadata, privacy, or compliance messages are logged and assigned.

## Internal Beta

- [ ] Tester invitations are delivered.
- [ ] TestFlight installation succeeds on iPhone and iPad.
- [ ] Real-device smoke test is completed and recorded.
- [ ] Crash feedback and tester submissions are monitored.
- [ ] Critical/high issues receive explicit ship/hold decisions.

## Response Timing

- Critical: same day
- High: within 24 hours
- Medium: next approved sprint
- Low: backlog

Use the App Store issue-response playbook and do not begin a hotfix until the issue is reproduced or
the required Apple correction is unambiguous.
