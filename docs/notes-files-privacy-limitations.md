# Notes and Files Privacy Limitations

The Notes & Files module is a testing/pre-launch prototype. It is not approved for real police records, evidence, statements, confidential information, sensitive personal information, official documents, or operational case material.

## Required User Warning

OPAi is currently in testing/pre-launch. Do not enter real police records, confidential information, sensitive personal information, real evidence, real statements, or official documents into this prototype.

Files in this prototype are metadata placeholders only. OPAi does not upload, store, process, or sync real files in this testing version.

## Local Storage Boundary

Sprint 013 uses Sprint 007 local persistence only. Local data may remain on the device until the user resets demo data, clears local data, uninstalls the app, or the platform removes app storage.

## Production Privacy Requirements

A future production Notes & Files implementation must support privacy-by-design, minimal data collection, consent controls, audit logging, secure retention, secure deletion, and documented data handling rules before real content is allowed.
