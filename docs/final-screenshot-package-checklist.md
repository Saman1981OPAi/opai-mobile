# Final Screenshot Package Checklist

Use screenshots from the exact release-candidate build and verify App Store Connect's currently
accepted dimensions before upload.

## Accepted iPhone 6.9-inch Sizes

- 1260 x 2736 portrait or 2736 x 1260 landscape
- 1290 x 2796 portrait or 2796 x 1290 landscape
- 1320 x 2868 portrait or 2868 x 1320 landscape

## Accepted iPad Sizes

- 2064 x 2752 portrait or 2752 x 2064 landscape
- 2048 x 2732 portrait or 2732 x 2048 landscape

## Recommended First Three

### iPhone

1. Home Dashboard
2. Start My Shift
3. Report Writing, OPAi Assistant, or Translation

### iPad

1. Dashboard overview
2. Calendar, Court, and Training workflow
3. OPAi Assistant or Notes & Files workflow

## Asset Review

- [ ] Exact accepted dimensions and PNG, JPG, or JPEG format
- [ ] Captured from the actual current UI
- [ ] Clear at normal display size with no clipped or overlapping content
- [ ] No real private data, police records, evidence, statements, or documents
- [ ] No official police/government marks or implied affiliation
- [ ] No copyrighted third-party imagery without permission
- [ ] No claims beyond actual local/mock functionality
- [ ] AI, legal, medical, and translation capabilities are not overstated
- [ ] Testing/pre-launch status remains clear where appropriate
- [ ] iPhone assets receive first priority; Android store assets remain paused
