# Incident Privacy Limitations

## Local Prototype Only

Sprint 010 stores incident drafts locally for prototype testing. It is not production storage and is not approved for real police records, evidence, confidential information, or sensitive personal information.

## Stored Locally

- Prototype incident basics
- Prototype persons involved
- Prototype witnesses
- Prototype notes
- Prototype attachment metadata
- Prototype status and priority
- Prototype follow-up flags

## Not Stored

- Real evidence files
- Real photos
- Real audio
- Real video
- Real documents
- Real biometric data
- Cloud documents
- Production police records
- Production RMS records
- Backend database records

## Attachment Metadata

Attachment entries are metadata only. Sprint 010 does not access camera, microphone, photo library, document picker, or cloud storage.

## Future Production Plan

Future production incident handling must use secure backend authentication, access controls, encryption, audit logging, privacy-by-design, explicit user consent, and Canadian privacy compliance planning before real operational data is accepted.
