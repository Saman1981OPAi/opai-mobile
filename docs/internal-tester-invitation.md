# Internal Tester Invitation

## Invitation Copy

Thank you for testing OPAi Police. This is an internal iOS beta/local prototype designed to
evaluate workflow, navigation, usability, reminders, local persistence, local notifications, and
App Store readiness.

Please do not enter real police records, confidential information, sensitive personal information,
real evidence, real statements, or official documents.

Please test the app using demo/mock information only and report bugs, confusing screens, layout
issues, broken buttons, missing disclaimers, or anything that feels unclear.

## Sending Feedback

Use TestFlight feedback when available, or email support@opaiapp.com. Include:

- Device model and iOS version
- Build number
- Screen or module
- Steps to reproduce
- Expected result and actual result
- Screenshot or screen recording when safe and useful

Do not include real operational, personal, evidentiary, or confidential information in feedback.
