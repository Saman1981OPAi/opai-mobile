# OPAi WhatsApp Channel

Build 22 adds an official OPAi WhatsApp Channel link in the app.

## URL

https://whatsapp.com/channel/0029Vb8HFSMEQIaoXOL6YO1a

## Placement

- Home Dashboard compact OPAi updates card.
- Settings Support screen.

## Guardrails

- Opens externally through HTTPS only.
- No tracking parameters are added.
- No in-app WhatsApp SDK is integrated.
- No messaging automation is connected.
- No user account or phone number is collected by the app.
