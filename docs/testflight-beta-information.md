# TestFlight Beta Information

## Beta App Description

OPAi Police is an iOS-first testing build of a local/offline productivity assistant for Canadian
policing workflows.

## What to Test

- Mock sign-in, create account, sign-out, and consent flow
- Home Dashboard and Start My Shift
- New Incident local drafting
- OPAi Assistant mock workflow and local history
- Translation mock workflow and local history
- Calendar, Court, Training, Requalification, and Follow-Up workflows
- Notes & Files metadata placeholders
- Settings, Privacy, Terms, and disclaimers
- Reset Demo Data and Clear Local Data
- Local notification permission, scheduling, and cancellation

## Beta Notes

Please use demo data only. Do not enter real police records, confidential information, sensitive
personal information, real evidence, real statements, or official documents.

## Known Limitations

- Local/offline prototype only
- Mock authentication; no real accounts
- No backend, production database, or cloud sync
- No OpenAI integration or real AI responses
- No real translation API, OCR, camera, microphone recording, or document translation
- No real file upload, evidence storage, or police-service/RMS integration
- No external calendar synchronization
- No payments or subscriptions
- Android release is paused pending the D-U-N-S Number for Ebrahimi Holdings
