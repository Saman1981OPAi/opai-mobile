# Public Release Notes

## Version 0.1.0-beta

Initial iOS testing release of OPAi Police.

This version includes a local and offline prototype for:

- Start My Shift reminders
- New Incident local drafts
- OPAi Assistant mock workflow
- Translation mock workflow
- Calendar, court, training, and requalification reminders
- Follow-up reminders
- Notes and file metadata placeholders
- Local notifications
- Privacy, legal, consent, and PTSD awareness screens

This build is intended for testing and demonstration only. Do not enter real police records,
confidential information, sensitive personal information, evidence, statements, or official
documents.

Target build `21` has not yet been generated or uploaded. Use these notes only after the uploaded
binary and App Store version are confirmed.
